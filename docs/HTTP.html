HTTP, Hypertext Transfer Protocol is essentially the language computers and servers use to talk to each other. Computers often request information and documents from servers using this language. This language is mostly used to execute get requests. Get requests are requests for the computer getting the information from a server. When the get request is executed the server the sends an HTML page to the user. The other side of the language is the post request, post request posts or sends the information you have typed out back to the server. HTTPS is essentially the same as HTTP, HTTPS is the secure protocol this insures not to let out any information to hackers and other people trying to get your data online.
